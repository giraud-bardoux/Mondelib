Module Flou (moduleflou) — v1.22 — SocialEngine PHP 7.4
-------------------------------------------------------
Installation :
1) Admin > Packages & Plugins > Package Manager > Upload le ZIP > Install
2) Aller sur /flou (démo) et /admin/flou (placeholder admin)
3) Vérifier engine4_core_modules: name=moduleflou, version=1.22, enabled=1

Usage du helper dans un .tpl :
  echo $this->flouThumb($photoUrl, 10, ['alt' => 'Miniature']);

Invité: flou appliqué • Connecté: image nette.
